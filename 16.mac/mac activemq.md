# mac activemq

安装
```
brew install activemq
```
运行
```
To have launchd start activemq now and restart at login:
  brew services start activemq
Or, if you don't want/need a background service you can just run:
  activemq start
```
管理地址
```
http://localhost:8161
```
访问地址
```
http://localhost:61616
```
